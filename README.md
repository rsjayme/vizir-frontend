### Desafio Vizir

Para rodar o código siga o passo a passo:

- Clone o repositório para sua máquina e vá para o diretório do desafio.

- Baixe as dependências do projeto.

  ```bash
  npm install

  # or

  yarn
  ```

- Rode o projeto e em seguida acesse: http://localhost:3000

  ```bash
  yarn dev

  # or

  npm run dev
  ```
